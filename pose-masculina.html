Como Usar

Envie a foto do homem como referência no Nano Banana e cole o prompt escolhido.

Referência

PERSON_A

Todos os prompts usam a imagem anexada como PERSON_A e preservam rosto, identidade e proporções.

Formato

4:5 / 9:16 / 1:1

4:5 para feed, 9:16 para story/Reels e 1:1 para foto de perfil.

Resultado

Ensaio realista

Use os prompts para retrato masculino, moda, lifestyle, fitness, marca pessoal e editoriais.
